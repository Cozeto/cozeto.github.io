# 博客网站搭建

<!-- [tag]: Web Application、Vite -->

<!-- [description]: 记录Blog网站搭建所用工具及流程 -->

简单记录一下搭建这个博客的流程。

之前其实很早就想搭一个类似的网站，但是因为种种原因耽搁了，比如说想搞出一个很复杂的成品，把我日常需求中能迁移到web应用上的都构建出来，结果处处碰壁最后半途而废了。现在趁着时间比较多，把目标放小一点简单地做一个博客网站出来。

最后整个流程还是非常简化的：

- 目标是搭建一个纯前端的静态博客，用Github Page部署，网站内容用文件( `.md` 和 `.pdf` )来管理，当作是静态资源进行更新。这个实现方案可以抛开证书、域名和公网服务器等等的需求。
- 需求给 `GPT` 让它写prompt，然后喂给 `gemini 3 flash` 搭的一个初始的框架，整体简单删改了一下之后下到本地手动又修改了一下，用vscode集成的 `copilot` 做了一些修改。
- 整体的技术栈框架是 `Vite + React + typescripts + tailwindcss` ，虽然好久没写前后端的东西了但是看还是看得懂的，用上面这一套流程搭一个网页自己改点小细节还是比较简单的。后面有什么idea都可以试着搞一搞。

最后效果看着还是挺满意的，以后用这个博客来记录一些学习笔记或者娱乐上的一些随笔等等。
